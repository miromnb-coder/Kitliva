# Kitliva

Kitliva is an Expo + React Native + TypeScript mobile app foundation for a used hobby-gear marketplace.

## Foundation v1

This first version contains:

- Expo SDK 54 foundation
- React Native + TypeScript
- Expo Router with tab navigation
- Kitliva theme colors and spacing
- Placeholder screens for Home, Search, Sell, Inbox, and Profile
- VS Code / Codeanywhere task buttons

## Run locally

Install dependencies:

```bash
npm install
```

Start Expo:

```bash
npm run start
```

Start Expo Web:

```bash
npm run web
```

Clear Expo cache:

```bash
npm run start:clear
```

Check TypeScript:

```bash
npm run typecheck
```

Check Expo dependencies:

```bash
npm run check:deps
```

Fix Expo dependencies:

```bash
npm run fix:deps
```
